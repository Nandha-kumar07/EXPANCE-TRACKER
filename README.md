# Expense Tracker App

This is a MERN stack application (MongoDB, Express, React, Node.js) to track expenses and income.

## Features
- Sign up and Login
- Add expenses and income
- View charts and reports
- Dark mode
- AI Chatbot to ask about your finances

## How to Run

### Backend
1. Go to backend folder: `cd backend`
2. Install dependencies: `npm install`
3. Create a `.env` file with your keys (MongoDB, Email, etc)
4. Run server: `npm run dev`

### Frontend
1. Go to frontend folder: `cd frontend`
2. Install dependencies: `npm install`
3. Run app: `npm run dev`

## Deployment
- Backend on Render
- Frontend on Vercel
